# 水墨 Theme

> 点墨成渊映竹影，化筠为简蕴幽香。
> 指尖经纬裁风雨，且护明眸夜未央。

一个包含多款风格化 VS Code 主题的配色集合，主打「护眼 + 格调 + 长时可读」。从中国传统水墨、丹青、竹简意象中取色，提供极佳的编码体验。

## 主题列表

- **竹简（浅色）** - *棕色系列*：基于完整的中国传统色彩体系与 WCAG AA 护眼标准构建。提供极致的阅读体验与大量 UI 级细节适配，适合白天长时沉浸编程。
- **水墨丹青（浅色）** - *水墨风系列*：取自中国传统水墨画色彩，淡雅素净，视觉柔和，为浅色环境下的长时间编码提供极佳的护眼体验。
- **竹简墨夜（深色）** - *棕色系列*：玄墨底色配合赭石与竹青的语义分工，具备极佳的对比度与层级感。适合夜间专注编码，有效缓解视觉疲劳。
- **月涌大江流（深色）** - *深蓝夜江系列*：深蓝夜江底色配合幽白银蓝的点缀，提供高层级感的暗色高对比度护眼主题。

## 主题特色

- **传统色护眼配色**：基于中国传统色提取冷暖调（如玄墨底色、竹青、赭石、黛蓝、胭脂），保证长时阅读的舒适与秩序感。
- **扩展语义高亮**：对大量现代语言环境（TS/JS/Vue/Python/Java/Go 等）进行了深度适配，补充 `readonly`、`decorator`、`defaultLibrary` 等语义 Token 映射。
- **全方位 UI 适配**：对 VS Code 整体 UI 界面（包括终端、Git、侧边栏、状态栏和搜索面板）进行细节打磨，保持极高的视觉统一性。

## 使用方式

1. 在 VS Code 扩展市场搜索并安装 `水墨 Theme`。
2. 打开命令面板（快捷键：`Cmd/Ctrl + Shift + P`）并执行 `Preferences: Color Theme`（首选项：颜色主题）。
3. 选择您需要的主题：**竹简**、**水墨丹青**、**竹简墨夜** 或 **月涌大江流**。

*快捷路径：`Cmd/Ctrl + K` -> `Cmd/Ctrl + T`。*
