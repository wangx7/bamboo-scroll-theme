# 水墨 Theme

> 点墨成渊映竹影，化筠为简蕴幽香。
> 指尖经纬裁风雨，且护明眸夜未央。

一个包含 4 款风格化 VS Code 主题的配色集合，强调「护眼 + 格调 + 长时可读」，灵感来自中国传统水墨、丹青与竹简。

## 主题列表

- **竹简（浅色）** - *棕色系列*：传统色体系 + WCAG AA 对比度，适合白天长时编码。
- **水墨丹青（浅色）** - *水墨风系列*：淡雅柔和，降低浅色环境下的视觉负担。
- **竹简墨夜（深色）** - *棕色系列*：玄墨底色搭配赭石与竹青语义分层，适合夜间专注。
- **月涌大江流（深色）** - *深蓝夜江系列*：深蓝底色配银蓝点缀，层级清晰、对比充足。

## 主题特色

- **传统色护眼配色**：基于中国传统色冷暖搭配，长时阅读更舒适。
- **扩展语义高亮**：完善 TS/JS/Vue/Python/Java/Go 等语义 Token 映射。
- **全方位 UI 适配**：统一优化终端、Git、侧边栏、状态栏和搜索面板。

## 使用方式

1. 在 VS Code 扩展市场搜索并安装 `水墨 Theme`。
2. 打开命令面板（快捷键：`Cmd/Ctrl + Shift + P`）并执行 `Preferences: Color Theme`（首选项：颜色主题）。
3. 选择您需要的主题：**竹简**、**水墨丹青**、**竹简墨夜** 或 **月涌大江流**。

*快捷路径：`Cmd/Ctrl + K` -> `Cmd/Ctrl + T`。*
